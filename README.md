# Chat-Hub
Source code for Chat Hub (chat.chippytime.com), a large language model hub created by Chippytech and uses OpenRouter it supports OpenAI's GPT models and sereval open source models.

The backend server uses a combination of the following tools and services: Streamlit + nginx + Google Cloud + OpenRouter.

Chat Hub is licensed under Apache 2.0 and uses a small google cloud server. Use the source as inspianation for your open/own projects.
